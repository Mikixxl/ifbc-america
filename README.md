# IFBC America LLC

Single-page website for IFBC America LLC — Florida administrative office in Miami's Brickell district providing treasury and settlement operations support for IFB International Finance Bank LTD.

## Stack

- Vanilla HTML / CSS / JS (no build step)
- 14-language i18n (EN, DE, FR, ES, IT, PT, RU, ZH, ZH-Hant, JA, KO, HI, AR, HE)
- Glassmorphism design (navy + gold)
- Cormorant Garamond (display) + Inter (sans)
- Deployed via Netlify

## Files

- `index.html` — single-page structure
- `styles-v1.css` — all styling (cache-bust via filename versioning)
- `script-v1.js` — i18n + modal logic
- `logo.png` (1280×1280), `hero.jpg` (1672×941), favicon set, app icons
- `site.webmanifest`, `robots.txt`, `sitemap.xml`, `netlify.toml`

## Domain

`ifbcamerica.com`

## Compliance

This is an administrative office. The company does NOT hold client funds, issue financial instruments, or operate as a bank, money transmitter, or clearing house. Banking instructions for any transaction are issued directly by IFB International Finance Bank LTD.
